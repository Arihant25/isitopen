# Is It Open?

A web application for checking the real-time status of canteens. Students can see which canteens are currently open, while canteen owners can update their status through a PIN-protected dashboard.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- MongoDB
- Tailwind CSS

## Usage

### For Students

Navigate to the home page and click "I want to Eat" to view the current status of all canteens.

### For Canteen Owners

1. Click "I am a Canteen" on the home page
2. Select your canteen
3. Enter your 4-digit PIN
4. Tap the screen to toggle between open and closed status
