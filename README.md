# Is It Open?

A web application for checking the real-time status of shops in IIIT Hyderabad. Students can see which shops are currently open, while shop owners can update their status through a PIN-protected dashboard.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- MongoDB
- Tailwind CSS

## Usage

### For Students

Navigate to the home page and click "I want to Eat" to view the current status of all shops.

### For Shop Owners

1. Click "I am a Shop" on the home page
2. Select your shop
3. Enter your 4-digit PIN
4. Tap the screen to toggle between open and closed status
